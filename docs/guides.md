# Guides

Comprehensive guides for using redis-ruby in production applications. These guides cover advanced topics, best practices, and detailed explanations of key features.

## Available Guides

Browse the guides in the navigation menu to learn about:

- **Connection Options** - Connecting to Redis with TCP, Unix sockets, TLS/SSL
- **Connection Pools** - Thread-safe and fiber-safe connection pooling
- **Pipelining** - Batching commands for improved performance
- **Transactions** - MULTI/EXEC and optimistic locking with WATCH
- **Pub/Sub Messaging** - Real-time messaging and event-driven architectures
- **Lua Scripting** - Server-side logic and atomic operations
- **Redis Sentinel** - High availability and automatic failover
- **Redis Cluster** - Automatic sharding and horizontal scaling
- **Client-Side Caching** - RESP3 tracking for improved performance
- **Distributed Locks** - Coordinating access to shared resources

Each guide includes detailed examples, best practices, and troubleshooting tips.
