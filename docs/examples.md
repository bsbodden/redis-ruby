# Examples

Practical examples demonstrating common use cases and patterns with redis-ruby. Each example includes complete, runnable code that you can use as a starting point for your own applications.

## Available Examples

Browse the examples in the navigation menu:

- **Basic Usage** - Fundamentals of connecting and working with Redis
- **Connection Pooling** - Thread-safe operations in multi-threaded applications
- **Pipelining** - Batching commands for dramatic performance improvements
- **Pub/Sub** - Real-time messaging and event-driven applications
- **Advanced Features** - Using JSON and Search for powerful use cases

Each example includes:
- Prerequisites and setup instructions
- Complete, runnable code
- Detailed explanations of key concepts
- Expected output
- Common pitfalls and how to avoid them
